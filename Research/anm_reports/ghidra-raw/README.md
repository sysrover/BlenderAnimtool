# Ghidra Raw Dumps

Put all future Ghidra raw outputs for this ANM/TXA/FBX research here.

Naming convention:

- `ghidra-raw-<topic>.txt` for text/decompile/script dumps.
- `ghidra-raw-<topic>.json` for schema, table, or structured HTTP outputs.

Markdown research files under `anm/` should reference these as
`ghidra-raw/<filename>`. Markdown files under subfolders such as `anm/fbx/`
should reference them as `../ghidra-raw/<filename>`.
